# 🚀 Smart Lib Terminal (SLT)
نظام ذكي لخدمة الأجهزة الضعيفة عبر توفير مساحة التخزين وتطوير "ذاكرة" للمكتبات البرمجية.

## 🛠 المميزات:
1. *Global Indexing:* البحث عن المكتبات في الجهاز بالكامل.
2. *Auto-Hunting:* نسخ المكتبات من البرامج الجديدة وحفظها في permanent_repo.
3. *Smart Linking:* دمج المكتبات في مشاريعك عبر (Symlinks) دون استهلاك مساحة (0MB).
4. *Auto-Clean:* مسح الروابط الوهمية فور إغلاق الـ Terminal ليبقى الجهاز نظيفاً.
5. *Compression:* ضغط المكتبات القديمة لتوفير مساحة القرص الصلب.

## 🚀 التشغيل:
1. قم بتثبيت المتطلبات: pip install watchdog
2. لتشغيل الواجهة والبحث: افتح main.py (كـ مسؤول).
3. للتشغيل الصامت مع الويندوز: استخدم start_system.vbs في مجلد Startup.

## 📁 هيكل المجلدات:
- data/permanent_repo: مستودع المكتبات الدائم.
- src/engine/watcher.py: المحرك المسؤول عن "صيد" المكتبات الجديدة.
